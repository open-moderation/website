# OpenModeration Website

Marketing website for OpenModeration, the open-source content moderation platform.

## Stack

- [Astro 6](https://astro.build)
- [Tailwind CSS 4](https://tailwindcss.com)
- Deployed at [www.openmoderation.com](https://www.openmoderation.com)

## Commands

```sh
pnpm install
pnpm dev      # localhost:4321
pnpm build    # production build to dist/
pnpm preview  # preview production build
```
