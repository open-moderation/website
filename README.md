The unified AI moderation platform. One API for every provider — OpenAI, Azure, Mistral, and more. Self-host or cloud, with automated workflows and DSA-compliant audit trails.

## What is OpenModeration?

The moderation market is fragmented. OpenAI, Azure, HuggingFace, Mistral — each has its own API format, category schema, and pricing. Integrating multiple providers means building and maintaining separate integrations for each one.

OpenModeration abstracts all of it. One API key, one format, one dashboard. Connect once and route content to the right provider based on language, type, and cost. Change providers with zero code changes.

## Why OpenModeration?

- **Single integration** — Connect once. Your moderation requests are automatically routed to the best provider for each piece of content.
- **BYOK (Bring Your Own Keys)** — Use your existing provider accounts. No markup, no hidden fees, full control over your costs.
- **Open source, fully auditable** — AGPL v3. Every line of code is public. No black boxes, no hidden logic.
- **Self-host or cloud** — Deploy on your own infrastructure with Docker for complete data sovereignty, or use our managed cloud for zero maintenance.
- **Built for compliance** — Every moderation decision logged with full audit trail. Configurable data retention. DSA and GDPR compliant by design.
- **Moderation dashboard** — Dedicated review interface for your Trust & Safety team. Manage flagged content, track decisions, and enforce your community guidelines.
